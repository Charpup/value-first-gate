# value-first-gate

Value-First Gate skill for OpenClaw.

## Purpose
Run a strict pre-implementation gate **after planning** and **before TDD/SDD**:
- Problem framing
- First-principles check
- Value scoring
- Go/No-Go decision

## Files
- `SKILL.md`
- `references/value-review-template.md`

## Install
Copy this folder into:
- `~/.openclaw/skills/value-first-gate/`
